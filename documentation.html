Avatar

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Example of Image Avatar

Avatar is available in 3 different sizes. You can use image in Avatar. You need to include class avatar and for size add class according to size avatar-lg, avatar-md, avatar-sm (e.g. class="avatar avatar-lg")
Don't forget to add Responsive Image class names for img element.

Images

Images can be responsive to fit the parent's width, and also can be customised to be round shaped

Example of Responsive Image

You can add class img-responsive to make your image fit the width of container. It's height will get adjusted by keeping the aspect ratio same. If you want to change the aspect ratio, you will have to crop the image.

Example of Active form fields

Add user-name-label class to labels and user-name-input for input element to have a username input.

Add password-label class to labels and basic-input for input element to have a password input.

Add basic-label class to labels and basic-input for input element to have a normal input.
